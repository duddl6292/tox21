모델 및 서비스 설명

본 서비스는 Tox21 데이터를 기반으로 화합물의 NR-ER 및 SR-p53 독성 위험도를 분석합니다.

분석 대상

본 프로젝트는 Tox21 데이터셋 중 NR-ER과 SR-p53 타깃을 사용합니다.

  • NR-ER: 에스트로겐 수용체 관련 활성 여부
  • SR-p53: p53 스트레스 반응 관련 활성 여부

입력 데이터

사용자는 SMILES 또는 화합물명을 입력할 수 있습니다. 화합물명 입력 시 PubChem API를 통해 SMILES로 변환한 뒤 분석합니다.

  • SMILES 입력: 바로 RDKit으로 구조 변환
  • 화합물명 입력: PubChem API로 SMILES 조회
  • 유효하지 않은 입력: 분석 불가 처리

특징 추출

분자 구조는 RDKit을 이용해 수치형 특징으로 변환됩니다.

  • Morgan Fingerprint
  • 2D Molecular Descriptor
  • Canonical SMILES 기반 DB 조회

분석 방식

입력 화합물이 Tox21 DB에 존재하면 실제 라벨을 우선 제공하고, DB에 없는 화합물은 학습된 머신러닝 모델로 예측합니다.

  • DB 실제 Tox21 라벨 사용
  • ML 모델 예측값 사용
  • INVALID 분석 불가 입력
