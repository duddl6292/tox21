분석 방식

DB 기존 Tox21 데이터에 존재하는 화합물은 실제 라벨을 출력합니다.

ML DB에 없는 신규 화합물은 FFNN 모델로 예측합니다.

위험도 기준
저위험: 0.0 ~ 0.3 중위험: 0.3 ~ 0.7 고위험: 0.7 ~ 1.0
입력 CSV 예시
name smiles 설명
ethanol CCO 이름 + SMILES
benzene c1ccccc1 이름 + SMILES
aspirin 이름만 입력
caffeine 이름만 입력
CC(=O)Oc1ccccc1C(=O)O SMILES만 입력

CSV 파일에는 smiles 또는 name 컬럼이 필요합니다.
두 컬럼이 모두 있는 경우 smiles 값을 우선 사용하고, smiles가 비어 있으면 name을 PubChem API로 SMILES 변환합니다.

잘못된 name, smiles가 포함된 경우 해당 행은 “분석불가”로 표시됩니다.
